## 📘 About the Project
The Simple To‑Do List App helps users manage tasks efficiently.  
It’s perfect for learning **HTML, CSS, and JavaScript** fundamentals while building something useful for everyday life.

---

## ⚙️ How It Works
- ✍️ **Add Task**: Type your task in the input box.  
- ➕ **Click Add**: The task appears in your list.  
- ✅ **Mark Done**: Check off tasks once completed.  
- 🗑️ **Delete Task**: Remove tasks you no longer need.  
- 🔄 **Update List**: The list refreshes dynamically.  
- 💾 **Save Progress**: Optionally store tasks using local storage.
